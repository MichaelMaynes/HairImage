# Hair Image Salon Website

This is the first public website draft for Hair Image in Titusville, Florida.

## Files

- `index.html` — public salon website
- `style.css` — all website styling
- `script.js` — mobile menu and temporary form behavior
- `admin.html` — placeholder for the future secure admin login
- `images/` — add salon and hairstyle photos here later

## Publish with GitHub Pages

1. Upload all files and the `images` folder to the root of your GitHub repository.
2. Open the repository's **Settings**.
3. Select **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`.
6. Save.

## Important

The booking, contact, review, and admin features are only visual in this first version.
Do not use the forms for real customer information until Supabase authentication and database security are connected.
